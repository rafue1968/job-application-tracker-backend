# Job Tracker

A simple application to track and manage job applications and opportunities.

## Features

- Track job applications with status updates
- Store job details (company, position, date applied)
- Monitor application timelines
- Organize opportunities by status (applied, interviewing, offered, rejected)

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
git clone https://github.com/yourusername/jobtracker.git
cd jobtracker
npm install
```

### Usage

```bash
npm start
```
